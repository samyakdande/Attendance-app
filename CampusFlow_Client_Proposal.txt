================================================================
4. RECURRING MONTHLY CLOUD COSTS (Variable based on usage)
================================================================
The ₹1,00,000 fee covers the one-time software development and engineering labor. It DOES NOT cover the monthly cloud infrastructure required to keep the app online (Database storage, Redis cache, Server hosting).

As the school adds more students, teachers, and attendance records, the storage and server usage will increase. You must choose how to handle these monthly bills:

Option A: Client Direct Billing (Recommended for Transparency)
You (the client) will create your own accounts on Supabase (Database), Upstash (Redis), and Render/AWS (Hosting) using your credit card. 
- The software will be handed over to your accounts.
- You will pay the cloud providers directly for exactly what you use.
- Estimated starting cost: ~$0 to $25/month (depending on initial user load).

Option B: Managed Hosting by Developer (Retainer + Infrastructure)
If you prefer not to manage cloud accounts, I will host the platform on my infrastructure.
- Base Maintenance Fee: ₹5,000 per month (covers my time for monitoring, backups, and minor bug fixes).
- Infrastructure Pass-Through: You will be billed additionally for the exact cloud costs incurred that month (e.g., if Supabase charges ₹2,000 for storage, that is added to the ₹5,000 base fee).
- Total Monthly Cost: ₹5,000 + Actual Cloud Usage.
